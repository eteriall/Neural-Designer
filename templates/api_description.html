The example request

Example POST request to neuraldesigners.com/api.

{
"api_key": "XXXX-XXXX-XXXX-XXXX",
"requested_service": "logo",
"add_svg_elements": True,
"rects_max_n": 2,
"poly_max_n": 4,
"color_style": "epic",
"sharpen": 0.5,
"image_size": (512, 512)
}

JSON Response contain errors, if some fields are missing, or some values are wrong. If operation is successful response contains JSON with svg image:

{"svg-text": ***svg-code***,
"svg-file": ***byte-array***}
